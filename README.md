# SnakeGame на WinAPI

## Описание проекта

SnakeGame — это классическая игра "Змейка", реализованная с использованием библиотеки WinAPI на языке C++. Ваша задача управлять змейкой, поедать пищу, избегая столкновений со стенами и самой собой.

## Установка

### 1. Сборка из исходников
Чтобы собрать проект из исходников, выполните следующие шаги:
1. Скачайте репозиторий с исходным кодом.
2. Откройте файл `.sln` в Visual Studio.
3. Соберите проект, выбрав конфигурацию (Debug или Release).
4. Запустите скомпилированный файл.

### 2. Скачивание бинарного файла
Для запуска игры вы можете скачать готовый бинарный файл из раздела [релизов](https://github.com/lTETHERl/SnakeGame-WinAPI/releases). Просто скачайте архив, распакуйте и запустите исполняемый файл.

## Инструкция по сборке в Visual Studio
1. Убедитесь, что у вас установлена Visual Studio с компонентами для работы с проектами на C++.
2. Скачайте исходный код из репозитория.
3. Откройте файл `.sln` в Visual Studio.
4. В Visual Studio выберите конфигурацию проекта (например, Debug или Release).
5. Нажмите кнопку **Собрать** (Build).
